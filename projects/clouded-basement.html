CloudedBasement.ca

A cloud hosting platform that automates Linux VPS provisioning, server configuration, and application deployment through the DigitalOcean API and GitHub workflows.

Role
Full-stack developer
Status
Live
Year

Platform

01 / Overview

Making server deployment more approachable.

CloudedBasement.ca reduces the manual work involved in preparing and deploying applications to a virtual private server. It brings provisioning and deployment steps into a more structured workflow while retaining the flexibility of Linux infrastructure.

02 / Engineering

From API request to configured server.

I engineered the VPS provisioning workflow using the DigitalOcean API and created GitHub webhook workflows for automated remote deployments. Server setup automates Nginx configuration, SSL, application runtimes, and deployment scripts so users do not have to repeat those steps manually.

  • Linux VPS provisioning
  • DigitalOcean API integration
  • GitHub webhook deployments
  • Nginx and SSL automation

03 / Outcome

A repeatable path from repository to server.

The platform packages several infrastructure tasks into one consistent process, reducing configuration errors and the time required to prepare a new deployment environment.
